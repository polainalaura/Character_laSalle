# Character_laSalle

Este proyecto es una aplicación de consola en Java que permite gestionar una lista de personajes. Cada personaje tiene un nombre, un nivel y una cantidad de experiencia. El programa ofrece un menú que permite crear nuevos personajes, eliminarlos, mostrar un ranking ordenado por experiencia y guardar o cargar los datos desde un archivo de texto. Se han incluido validaciones para asegurar que el usuario introduce datos correctos y para garantizar que las acciones se realizan únicamente cuando es posible.
